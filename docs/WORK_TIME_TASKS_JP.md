# 作業時間記録アプリ 開発タスク

## 基本機能
- [ ] 開始/終了ボタンで作業を記録するタイマーを実装
- [ ] 新しい記録開始時に前の記録を自動終了
- [ ] 開始/終了時刻・作業内容を手動で編集可能にする

## 記録項目
- [ ] プロジェクト管理番号とプロジェクト名の入力・自動補完
- [ ] 作業カテゴリの選択（ドロップダウンまたはフリーワード）
- [ ] 作業内容入力欄
- [ ] 開始時刻・終了時刻の自動入力と手動編集

## レポート
- [ ] 日付を選択し、記録を日別に一覧表示
- [ ] プロジェクト管理番号 × 作業カテゴリで合計時間を集計
- [ ] 作業内容ごとの内訳を「作業内容（合計時間）」形式で表示

## 保存・データ管理
- [ ] 保存先ディレクトリを設定可能にする
- [ ] JSON または SQLite でデータ保存
- [ ] 日付・プロジェクトでフィルタできるUI

## 技術選定
- [ ] デスクトップ版は Tauri + Svelte を利用
- [ ] 将来的に PWA 対応も検討

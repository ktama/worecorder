# Worecorder

このプロジェクトは **Tauri** と **Svelte** フロントエンドを使用しています。デスクトップアプリを構築するための基本的な足場を提供します。

## プロジェクトの目的

Worecorder は軽量なデスクトップ用作業記録アプリを目指しています。Web ベースのユーザーインターフェイスと Tauri が提供するネイティブ機能を組み合わせています。現在は作業時間を記録する機能のほか、記録の編集やプロジェクト別フィルタ表示にも対応しており、簡易的なタイムトラッカーとして利用できます。

## セットアップ方法

依存関係をインストールし、開発サーバーを起動します。

```bash
npm install
npm run dev
```

Tauri ウィンドウでアプリケーションを実行するには次を使用します。

```bash
npm run tauri
```

## 開発ガイドライン

- ブランチ名は日本語にすること。
- コードのフォーマットを統一し、わかりやすいコミットメッセージを書くこと。
- 変更をプッシュする前に、フロントエンドと Tauri バックエンドの両方をテストすること。
- プルリクエストは一つの論理的な更新に集中させること。

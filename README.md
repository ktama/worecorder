# Worecorder

このプロジェクトは **Tauri** と **Svelte** を使ったデスクトップ向け作業時間記録アプリです。
ローカル環境で完結し、開始/終了ボタンによる直感的な記録と日別集計を行えます。

## プロジェクトの目的

個人が手軽に作業時間を管理できるツールを提供します。クラウド連携は行わず、データはすべてローカルに保存されます。

## セットアップ方法

依存関係をインストールし、開発サーバーを起動します。

```bash
npm install
npm run dev
```

Tauri ウィンドウでアプリケーションを実行するには次を使用します。

```bash
npm run tauri
```

## 開発ガイドライン

- コードのフォーマットを統一し、わかりやすいコミットメッセージを書くこと。
- 変更をプッシュする前に、フロントエンドと Tauri バックエンドの両方をテストすること。
- プルリクエストは一つの論理的な更新に集中させること。

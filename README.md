# Worecorder

このプロジェクトは **Tauri** と **Svelte** フロントエンドを使用しており、デスクトップアプリを構築するための基本的な足場を提供します。

## プロジェクトの目的

Worecorder は軽量なデスクトップ用音声レコーダーを目指しています。Web ベースのユーザーインターフェイスと Tauri が提供するネイティブ機能を組み合わせています。今後は音声ストリームの取得と録音をローカルに保存する機能を中心に開発していきます。

## セットアップ方法

依存関係をインストールし、開発サーバーを起動します。

```bash
npm install
npm run dev
```

Tauri ウィンドウでアプリケーションを実行するには次を使用します。

```bash
npm run tauri
```

## 開発ガイドライン

- コードのフォーマットを統一し、わかりやすいコミットメッセージを書くこと。
- 変更をプッシュする前に、フロントエンドと Tauri バックエンドの両方をテストすること。
- プルリクエストは一つの論理的な更新に集中させること。
